# Self-Writing Software

Software that improves itself.

Production signals flow in. Diagnosed fixes flow out as PRs. Agents review agents. Humans approve what matters and teach the system what it can handle on its own. The loop closes.

## The idea

Every production error, every analytics shortfall, every piece of customer feedback is a request from the system to be improved. Self-writing software treats it that way — consolidating signals, generating fixes, and surfacing only the decisions that need a human.

The human's job isn't to review code. It's to set direction.
